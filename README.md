# React Portfolio (Demo)

Small React + GraphQL demo app intended for showcasing on a résumé.

Features
- Vite + React 18
- Apollo Client for GraphQL queries
- MSW (Mock Service Worker) to mock a GraphQL endpoint locally

Run locally

```bash
cd react-portfolio
npm install
npm run dev
```

Notes
- The GraphQL endpoint is mocked via MSW during development.
- Open http://localhost:5173 and inspect the mocked GraphQL responses.
